## Phase 2 — Video Editor overhaul (the biggest single chunk)                                                                                                                                                                       
Goal: turn the editor at `/episodes/:id/edit` into something a solo creator can actually finish a Short in without bailing to DaVinci.                                                                                              
                                                                                                                                                                                                                                    
Audit the current editor first and produce `docs/editor-audit.md` listing what exists, what's stubbed, and what's missing, then implement against the list below. Treat this as its own multi-PR sub-phase.                         
                                                                                                                                                                                                                                    
### Timeline & playback                                                                                                                                                                                                             
- Multi-track timeline: video (scenes), captions, voice, music, SFX, overlay (text/B-roll) — independently lockable/muteable/soloable                                                                                               
- Frame-accurate scrubbing with a virtualised renderer (windowed track rendering — must handle a 10-minute, 50-shot project at 60fps scrub on iGPU)                                                                                 
- Snapping (to playhead, clip edges, markers, beat grid if music is loaded)                                                                                                                                                         
- Ripple, roll, slip, slide edits — all with modifier-key variants matching Premiere/Resolve conventions where possible                                                                                                             
- Razor tool (S), select (V), trim-to-playhead, blade-all-tracks                                                                                                                                                                    
- J/K/L transport, I/O for in/out points, space to play/pause, comma/period for frame step                                                                                                                                          
- Zoom: Ctrl+scroll (timeline), Alt+scroll (track height), `\` to fit project                                                                                                                                                       
- Markers (M to add, Shift+M to add with note), marker list panel                                                                                                                                                                   
- Timeline minimap                                                                                                                                                                                                                  
                                                                                                                                                                                                                                    
### Editing model                                                                                                                                                                                                                   
- Non-destructive: every operation produces a new revision in an in-memory edit graph; serialise to disk on autosave                                                                                                                
- Undo/redo with a visible history panel (50 step minimum, infinite within session)                                                                                                                                                 
- Autosave every 10s + every significant action; recover-on-crash dialog on next launch                                                                                                                                             
- Project snapshots ("Save version" button) → restore-to-version with diff preview                                                                                                                                                  
                                                                                                                                                                                                                                    
### Clip operations                                                                                                                                                                                                                 
- Speed/time remap (0.25× to 4×) with optional pitch preservation                                                                                                                                                                   
- Audio: per-clip volume + gain automation envelopes (drag points on a curve)                                                                                                                                                       
- Audio: ducking already exists (sidechain) — surface it on the music track as a visible automation curve, not just a setting                                                                                                       
- Waveform rendering on audio tracks (compute once, cache)                                                                                                                                                                          
- Per-clip filters: brightness/contrast/saturation, basic LUT slot                                                                                                                                                                  
- Per-clip transform: position/scale/rotation/opacity with keyframes                                                                                                                                                                
- Transitions: cut (default), fade, dip-to-color, cross-dissolve, slide, push — drag from a panel onto a clip boundary                                                                                                              
                                                                                                                                                                                                                                    
### Captions                                                                                                                                                                                                                        
- Inline caption editing on the timeline (click a caption clip → edit text in place)                                                                                                                                                
- Style presets (already exist) but with live preview and a "save as preset" action                                                                                                                                                 
- Multi-line break with Shift+Enter; speaker colour-coding driven by the existing speaker tags                                                                                                                                      
                                                                                                                                                                                                                                    
### Scenes panel                                                                                                                                                                                                                    
- Drag-and-drop reorder                                                                                                                                                                                                             
- Quick swap: right-click a scene → "Regenerate" (reuses prompt) or "Edit prompt"                                                                                                                                                   
- Variations gallery: keep N=4 best variants per scene with one-click swap                                                                                                                                                          
                                                                                                                                                                                                                                    
### Render & export                                                                                                                                                                                                                 
- Render presets: YouTube Shorts (1080×1920, 30/60fps, h264 + aac), YouTube Long-form (1920×1080), TikTok (matching Shorts), audio-only (mp3/wav for the audiobook flow)                                                            
- Render queue with cancel, retry, render-region (from in to out point)                                                                                                                                                             
- Pre-render a fast proxy while the user works; final render uses source media                                                                                                                                                      
                                                                                                                                                                                                                                    
### Polish                                                                                                                                                                                                                          
- Empty state: a 60-second built-in tour the first time the editor opens                                                                                                                                                            
- Loading skeletons for every panel                                                                                                                                                                                                 
- Performance: virtualise the scene gallery; lazy-load thumbnails; use `requestAnimationFrame` for playhead, not React state                                                                                                        
                                                                                                                                                                                                                                    
Acceptance: the 50-shot/10-minute benchmark passes; a brand-new user can trim, reorder, add a transition, and render in under 5 minutes with no docs.
