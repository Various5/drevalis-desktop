## Phase 1 — Information Architecture refactor                                                                                                                                                                                      
Goal: a sidebar a first-time solo creator can navigate without help, and a Settings page that doesn't sprawl.                                                                                                                       
                                                                                                                                                                                                                                    
### Sidebar regrouping (rename `Sidebar.tsx` sections)                                                                                                                                                                              
- **Create**: Dashboard, Series, Episodes, Audio Studio (renamed from "Text to Voice"), Characters (renamed from "Character Packs"), Assets, Templates (promoted out of Settings → Content)                                         
- **Publish**: Calendar, Channels (new unified hub, see below)                                                                                                                                                                      
- **Monitor**: Jobs, Usage, System Log (renamed from "Event Log")                                                                                                                                                                   
- **Maintenance** (new, replaces parts of Settings → System): Health, Storage, Backup, Updates, FFmpeg, Diagnostics                                                                                                                 
- **Bottom**: Settings, Help                                                                                                                                                                                                        
                                                                                                                                                                                                                                    
### Replace conditional platform items with a Channels hub                                                                                                                                                                          
Kill the conditional sidebar items (YouTube, TikTok, Instagram, Facebook, X). Instead, ship `/channels`:                                                                                                                            
- Always-visible cards for every supported platform (YouTube, TikTok, Instagram, Facebook, X)                                                                                                                                       
- Disconnected → "Connect" CTA with inline OAuth flow                                                                                                                                                                               
- Connected → status pill (last publish, queued count, channel name), "Manage" button deep-linking to the existing per-platform detail route                                                                                        
- Keep the per-platform routes (`/youtube`, `/social/:platform`) so deep links survive; just unify the entry point                                                                                                                  
- This fixes the discoverability problem (today, the only way to learn TikTok integration exists is to find it buried in Settings → Integrations → Social Media)                                                                    
                                                                                                                                                                                                                                    
### Settings restructure                                                                                                                                                                                                            
Remaining Settings sections after the moves above:                                                                                                                                                                                  
- **Account** (License; Team / 2FA / Login history only on web build)                                                                                                                                                               
- **Appearance** (Theme — expand with density toggle: compact / comfortable; reduced motion)                                                                                                                                        
- **Privacy** (Crash reporting)                                                                                                                                                                                                     
- **Integrations** (LLM Configs, ComfyUI Servers, Voice Profiles, Social Media, API Keys)                                                                                                                                           
- **Network** (LAN API Access — see Phase 4 for treatment)                                                                                                                                                                          
                                                                                                                                                                                                                                    
### Naming changes (codemod across i18n keys + route titles)                                                                                                                                                                        
- Text to Voice → **Audio Studio** (it also handles music; current name under-sells)                                                                                                                                                
- Event Log → **System Log**                                                                                                                                                                                                        
- Character Packs → **Characters**                                                                                                                                                                                                  
- "Cloud GPU" → keep, but hide on desktop build via existing platform check (already done — verify it's still working after the regrouping)                                                                                         
                                                                                                                                                                                                                                    
### Mobile bottom nav                                                                                                                                                                                                               
Reduce to: Create · Publish · Monitor · Settings (4 tabs). Sub-items via a sheet.                                                                                                                                                   
                                                                                                                                                                                                                                    
Acceptance: every nav action reachable in ≤2 clicks; first-run sidebar with zero connected channels is informative, not empty.
