## 0. Route table (`frontend/src/App.tsx`)

Three layout wrappers:

- **No layout** (public): `/login`, `/reset-password` — outside `LoginGate`/`LicenseGate` (`App.tsx:125-127`).
- **`EditorLayout`** (fullscreen, no sidebar/header), behind `LoginGate` (`App.tsx:130-139`).
- **`Layout`** (sidebar + header), behind `LoginGate` (`App.tsx:140-170`).

Whole tree is wrapped by `LicenseGate` (`App.tsx:121`); the `Layout`/`EditorLayout`
groups are additionally wrapped by `LoginGate` (`App.tsx:132,142`).

## 2. Sidebar (`components/layout/Sidebar.tsx`)

Collapsible left rail: **240px expanded / 56px collapsed** (`Sidebar.tsx:169`).
Hidden `< md` (`Sidebar.tsx:168`). Logo "Drevalis" at top (`Sidebar.tsx:173-180`).
Order = workflow frequency (`Sidebar.tsx:49`).

**Top** (`NAV_TOP`, `Sidebar.tsx:52-54`)
1. Dashboard → `/` — icon `LayoutDashboard`.

**Section: Content Studio** (`NAV_CONTENT_STUDIO`, `Sidebar.tsx:57-63`)
2. Episodes → `/episodes` (`Film`) — **shows a generating-count badge** (see §8).
3. Series → `/series` (`Layers`).
4. Text to Voice → `/audiobooks` (`Mic`).
5. Assets → `/assets` (`FolderOpen`).
6. Character Packs → `/character-packs` (`Users`).

**Section: Publish** (`NAV_PUBLISH_STATIC`, `Sidebar.tsx:69-71`)
7. Calendar → `/calendar` (`CalendarDays`) — always shown.
8. **YouTube** → `/youtube` (`Youtube`) — **conditional:** only when `youtubeConnected` (`Sidebar.tsx:246-252`).
9. **TikTok / Instagram / Facebook / X** → `/social/:platform` — **conditional:** each renders only if that platform is in `connectedSocials` (`Sidebar.tsx:253-265`). Platform→icon/label map at `Sidebar.tsx:37-46` (`tiktok`=Music2, `instagram`=Instagram, `facebook`=Facebook, `x`=Twitter).

Connection state comes from `useConnectedPlatforms()` (`Sidebar.tsx:153`), backed by
`/api/v1/social/platforms` + `/api/v1/youtube/status` (per file comment `Sidebar.tsx:32-36`).

**Section: System** (`NAV_SYSTEM_FULL`, `Sidebar.tsx:78-84`; filtered → `NAV_SYSTEM`, `Sidebar.tsx:86-88`)
10. Settings → `/settings` (`Settings`).
11. **Cloud GPU** → `/cloud-gpu` (`Cpu`) — **removed from sidebar when `isTauri()`** (`Sidebar.tsx:86-88`); page still reachable directly.
12. Jobs → `/jobs` (`ListChecks`).
13. Usage → `/usage` (`Activity`).
14. Event Log → `/logs` (`Terminal`).

**Bottom** (separated by divider, `NAV_BOTTOM`, `Sidebar.tsx:90-92`)
15. Help → `/help` (`HelpCircle`).

**Collapse toggle** at the very bottom (`Sidebar.tsx:281-300`). Collapsed mode shows
icons only; section headers collapse to a single icon (`Sidebar.tsx:107-118`); labels
become `title` tooltips.
