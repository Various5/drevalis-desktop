Install on your computer

Install in one command.

Paste a single line into your terminal. The installer downloads everything, sets it up, and starts it for you. About 5 minutes start to finish.

Linux · server or desktop

One line, headless or GUI

Just needs Docker Engine + the Compose plugin. Works on Ubuntu, Debian, Fedora, Arch — anything with docker and docker compose on PATH. SSH into your VPS or workstation and paste:

curl -fsSL https://drevalis.com/install.sh | bash

Prefer to read before piping? View install.sh.

No Docker yet? (Ubuntu / Debian) 
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
newgrp docker

Log out and back in if newgrp isn't enough. Then rerun the install command above.

macOS · Apple silicon & Intel

One line on Mac

Install Docker Desktop for Mac first (free, includes Compose). Then open Terminal and paste:

curl -fsSL https://drevalis.com/install.sh | bash

View install.sh.

Windows 10 · 11

One line in PowerShell

Install Docker Desktop first (it enables WSL2 automatically). Then open PowerShell and paste:

iwr -useb https://drevalis.com/install.ps1 | iex

If PowerShell blocks remote scripts, run Set-ExecutionPolicy -Scope Process RemoteSigned first. View install.ps1.

First time with Docker Desktop?
  1. Download Docker Desktop from docker.com — choose Windows · AMD64 or ARM64.
  2. Run the installer, accept the WSL2 option when prompted.
  3. Reboot when it asks.
  4. Launch Docker Desktop; wait until the tray icon is solid, not animated.
  5. Open PowerShell and paste the command above.

What the installer actually does

  1. 1Creates /srv/drevalis (Linux) or C:\drevalis (Windows).
  2. 2Writes docker-compose.yml pulling the official images from ghcr.io/drevaliscs/*.
  3. 3Creates a unique encryption key just for your install (it stays on your computer — we never see it).
  4. 4Pulls all images (~3 GB — coffee break).
  5. 5Runs database migrations.
  6. 6Starts the stack and prints the URL to open.

System requirements

Minimum

Works, some things slow

  • — Linux (Docker Engine), macOS (Docker Desktop), or Windows (WSL2 + Docker Desktop)
  • — 8 CPU cores, 16 GB RAM
  • — 50 GB free disk space
  • — Docker Engine 24+, Compose v2
  • — GPU optional (can offload to RunPod on Pro tier)
Recommended

Smooth generation on any format

  • — 12+ CPU cores, 32 GB RAM
  • — NVIDIA GPU with 12 GB+ VRAM
  • — 200 GB NVMe SSD
  • — 100 Mbps+ internet for first-time model downloads

See the hardware matrix on the homepage for end-to-end timing per build.

After install

  1. 1Open http://localhost:3000.
  2. 2Paste your license key (from the purchase email) into the activation screen.
  3. 3Settings → ComfyUI Servers → add your ComfyUI URL (e.g. http://host.docker.internal:8188).
  4. 4Settings → LLM Configs → add your LM Studio / Ollama / Claude endpoint.
  5. 5Settings → Voice Profiles → pick a default TTS voice.
  6. 6Create your first series and generate an episode.

Updates

In-app, one click: Settings → Updates → Update now. The updater sidecar pulls the new images and restarts the stack automatically. No terminal needed.

Prefer manual?

cd /srv/drevalis
docker compose pull
docker compose up -d --remove-orphans

Manual install (if you'd rather not pipe to bash)

Create a docker-compose.yml with:

services:
  app:
    image: ghcr.io/drevaliscs/creator-studio-app:latest
    restart: unless-stopped
    ports: ["8000:8000"]
    env_file: .env
    volumes: [./storage:/app/storage, updater_shared:/shared]
    depends_on: [postgres, redis]

  worker:
    image: ghcr.io/drevaliscs/creator-studio-app:latest
    restart: unless-stopped
    env_file: .env
    volumes: [./storage:/app/storage]
    command: python -m arq drevalis.workers.settings.WorkerSettings
    depends_on: [postgres, redis]

  frontend:
    image: ghcr.io/drevaliscs/creator-studio-frontend:latest
    restart: unless-stopped
    ports: ["3000:3000"]
    depends_on: [app]

  postgres:
    image: postgres:16-alpine
    restart: unless-stopped
    environment:
      POSTGRES_USER: drevalis
      POSTGRES_PASSWORD: drevalis
      POSTGRES_DB: drevalis
    volumes: [pg:/var/lib/postgresql/data]

  redis:
    image: redis:7-alpine
    restart: unless-stopped

  updater:
    image: ghcr.io/drevaliscs/creator-studio-updater:latest
    restart: unless-stopped
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - .:/project
      - updater_shared:/shared

volumes:
  pg:
  updater_shared:

Then write .env with ENCRYPTION_KEY (generate with python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())") and LICENSE_SERVER_URL=https://license.drevalis.com, then run docker compose up -d.
